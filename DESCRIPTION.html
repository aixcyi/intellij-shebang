This plugin provides Shebang insertion and management.
提供预置 Shebang 的插入和管理。

Shebang is also called:
Shebang 的别名有：
hashbang／sha-bang／pound-bang／hash-pling／sharp-exclamation

  • Insert a preset Shebang into file head.
    Menu：Code → Generate... → Shebang...
  • Manage preset Shebangs.
    Menu：File → Settings... → Tools → Shebang
  • 在文件头部插入 Shebang
    菜单：代码 → 生成... → Shebang...
  • 管理预置的 Shebang
    菜单：文件 → 设置... → 工具 → Shebang
