This plugin provides Shebang (also called Hashbang) insertion and management.

  • Insert a preset Shebang into file head.
    Menu：Code → Generate... → Shebang...
  • Manage preset Shebangs.
    Menu：File → Settings... → Tools → Shebang

提供预置 Shebang（也叫 Hashbang）的插入和管理。

  • 在文件头部插入 Shebang
    菜单：代码 → 生成... → Shebang...
  • 管理预置的 Shebang
    菜单：文件 → 设置... → 工具 → Shebang
